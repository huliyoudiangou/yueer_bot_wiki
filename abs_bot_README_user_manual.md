## 第一章：踏入仙途，账号与基础管理

私聊发送 `/start` 即可打开用户主菜单。

主菜单包含：`我的档案`、`每日修行`、`聚宝斋`、`药园`、`修仙`、`宗门`、`活动`、`求书中心`、`资产交易`、`账号服务`。

### 注册账户

入口：`账号服务 -> 注册账户`

可发送：

```text
注册账户
使用邀请码
邀请码
```

流程：

```text
1. 输入想要的用户名
2. 输入邀请码
3. 设置安全码 PIN
4. 设置 ABS 登录密码
```

用户名要求：3-20 位，只能包含字母、数字、下划线。注册成功后，Bot 会创建 ABS 账号，并绑定你的 Telegram ID。若开户或本地档案写入失败，Bot 会尽量退回邀请码；如果提示邀请码退回失败或需要管理员核查，请不要重复尝试同一邀请码，先联系管理员处理。

邀请注册有效期：默认按当前系统配置获得 30 天有效期。若同一 Telegram 用户曾经注册过并保留本地档案，再次使用邀请码注册时，账号有效期至少会补到本次注册默认有效期；若旧有效期更长，则保留更长的旧有效期，不叠加、不缩短。

### GitHub 福利

入口：`活动 -> GitHub福利`，或私聊发送 `github福利`

适用于 GitHub 注册满 5 年的用户。发送 `github福利` 后，Bot 会生成一个仅属于你本次领取流程的校验码，有效期 1 小时。你需要把这段校验码临时放入 GitHub 个人资料 Bio，然后回到私聊发送：

```text
校验github福利 GitHub用户名
```

校验通过后，Bot 会按当前 Telegram 账号状态发放奖励：未绑定 ABS 账号时发 1 个邀请码；已绑定 ABS 账号时发 150 天续期卡。每个 Telegram 账号最多领取一次，每个 GitHub id 最多领取一次；活动名额有限，是否开放和剩余名额以 Bot 当场提示为准。

### 个人邀请拉新

炼气初期及以上的正式账号，私聊发送以下任意命令可生成或查看个人邀请链接：

```text
我的邀请
邀请链接
拉新链接
```

新人通过邀请链接注册后获得 7 天体验权限；体验期内累计听书满 10 小时后，私聊发送 `新人任务` 可领取 7 天体验延期。邀请者每产生 1 名有效新人获得 10 积分，每月最多 150 积分；同一邀请链接每天最多激活 3 名新人体验。新人体验账号不能生成邀请链接，需要先使用正式邀请码转正。


流程：输入已有 ABS 用户名、输入 ABS 密码、设置安全码 PIN。绑定后，Bot 会把你的 Telegram ID 与已有 ABS 账号关联。

### 使用续期卡

入口：`账号服务 -> 使用续期卡`

可发送：

```text

```

流程：发送续期卡密，Bot 校验卡密，校验成功后为账号续期。若账号已因到期暂停，续期后会自动尝试恢复 ABS 收听权限；如果提示恢复权限或本地状态同步异常，请联系管理员人工核查。

入口：`账号服务 -> 获取线路`

可发送：

```text

```

用于获取当前 ABS 有声书服务器连接地址。

### 账号安全

入口：`账号服务 -> 账户安全`

可发送：

```text
修改密码
仅解绑不删号
解绑
删号注销
注销
```

说明：

- `修改密码`：需要先输入安全码，再输入新密码。
- `仅解绑不删号`：需要安全码，只解除 Telegram 与账号绑定，不删除 ABS 账号。
- `删号注销`：高危操作，需要安全码和二次确认 `确认注销`。该操作会删除本地记录，并在满足条件时删除 ABS 账号；若删除过程中提示本地档案或审计写入失败，请联系管理员人工核查。
- 特殊权限账号如无法自助注销，请联系服主处理。

可发送：`刷新我的今日净修为`

说明：按北京时间当天统计 ABS 听书记录，用于刷新今日宗门任务和个人档案中的净修为。该命令需私聊执行。

### 播放异常风控

后台每日 03:00 后按北京时间完整自然日统计 ABS 原始播放时长，检测目标为昨天和前天，不使用天道压制后的净修为。风控从系统记录的上线生效日起算，生效日前的历史收听时长既往不咎，不参与连续两日判断。

规则：

```text
单日原始播放时长 > 15 小时：Bot 私聊提醒一次
连续两个自然日 > 15 小时：临时暂停 ABS 听书权限 24 小时
```

触发暂停时，Bot 会私聊说明异常日期、播放时长和预计恢复时间，并在公告群提示账号因播放异常暂停一天。本次风控不扣除积分、贡献、声望或修为。

暂停满 24 小时后后台自动恢复。若账号已到期、已被管理员封禁或绑定的 ABS ID 已变化，系统不会自动恢复，避免误解封。

## 第五章：听书与修仙大盘

听书不仅是娱乐，也是修仙系统的核心修为来源。随着 ABS 听书时长增加，你的境界会不断提升。

### 境界体系

当前境界路线：凡人、炼气、筑基、结丹、元婴、化神。每个大境界内部有初期、中期、后期、圆满。

默认大境界修为范围：

```text
凡人：0-10 小时
炼气：10-50 小时
筑基：50-150 小时
结丹：150-350 小时
元婴：350-750 小时
化神：750 小时以上
```

默认小阶段阈值：

```text
炼气：初期 10，中期 20，后期 30，圆满 40
筑基：初期 50，中期 75，后期 100，圆满 125
结丹：初期 150，中期 200，后期 250，圆满 300
元婴：初期 350，中期 450，后期 550，圆满 650
化神：初期 750，中期 900，后期 1050，圆满 1200
```

### 大境界突破

入口：`修仙 -> 突破`

可发送：`突破`

流程：达到当前境界大圆满后发送 `突破`，Bot 检查修为、冷却、所需丹药和积分。有丹药时可使用背包丹药；无丹药但积分足够时可自动代购并渡劫。按提示回复 `确认渡劫` 或 `确认代购并渡劫`；如果在群内发起突破，可直接在群内回复确认词。

默认突破规则：

```text
凡人 -> 炼气：最低 10 小时，引灵入体，消耗 0 积分，成功率 100%，冷却 0 小时
炼气 -> 筑基：最低 50 小时，筑基丹，消耗 100 积分，成功率 80%，冷却 24 小时
筑基 -> 结丹：最低 150 小时，降尘丹，消耗 200 积分，成功率 60%，冷却 48 小时
结丹 -> 元婴：最低 350 小时，九曲灵参丹，消耗 500 积分，成功率 40%，冷却 72 小时
元婴 -> 化神：最低 750 小时，补天丹，消耗 1000 积分，成功率 20%，冷却 168 小时
```

渡劫补充规则：

```text
连续失败满 3 次后，下次突破触发保底
突破成功返还本次积分消耗的 20%
突破失败最多扣除 50 积分调养费，余额不足时按当前余额扣除
筑基及以上突破失败时，雷劫外溢可能随机波及 3 名普通用户
雷劫外溢每名受影响用户扣除 10 积分医疗费
```

风险：突破有成功率；失败会增加渡劫失败次数；高境界失败可能触发雷劫外溢；刚突破后存在巩固期，期间不能再次突破。修仙配置可随活动调整，具体以 Bot 当前提示为准。

### 修仙榜

入口：`修仙 -> 修仙榜`

可发送：


### 上架商品

可发送：`上架商品`

上架方式：

```text
1. 自由上架
2. 从背包上架
```

自由上架流程：发送 `上架商品`，达到筑基期及以上后可回复 `1` 或 `自由上架`，输入商品名称，输入商品价格，逐行发送卡密。

自由上架限制：仅筑基期及以上修士可用；商品名称 2-40 个字，不能包含换行、制表符或控制字符；价格 1-100000 积分；最多 50 行卡密；每条卡密最多 200 个字，且不能包含制表符或其他控制字符。适合卡密、邀请码、续期卡等一行一个的密钥类商品。

从背包上架流程：发送 `上架商品`，回复 `2` / `从背包上架` / `背包上架`，输入乾坤袋物品名，输入上架数量，输入单价。

背包上架限制：物品名 1-40 个字，不能包含换行、制表符或控制字符；上架数量最多 1000；单价 1-100000 积分。适合丹药、灵草、种子等乾坤袋物品。背包上架会先锁定卖家库存，成交后自动转入买家乾坤袋。

上架成功后，Bot 会向群内同步一条交易行新货提醒，展示商品 ID、类型、名称、单价、库存、48 小时有效期和购买命令；丹药类商品会附带简要功效，卡密明文不会出现在群内。

交易行商品有效期为 48 小时。到期后 Bot 会自动下架；背包商品未售库存会退回卖家乾坤袋，自由卡密商品未售卡密不再出售。功能上线前已经在售的商品，会从 Bot 上线该功能后开始 48 小时倒计时。

若商品卖家数据出现不一致，系统会暂停该商品交易并通知管理员核查，不会继续自动下架、购买成交或错误退回库存。

### 购买商品

可发送：

```text
购买商品 商品ID
购买商品 商品ID 数量
```

数量不填默认为 1，单次购买数量最多 50。自由卡密商品成交后，Bot 私聊发送卡密；背包物品商品成交后，自动进入买家乾坤袋。高额订单会要求回复 `确认购买商品`。

### 卖家管理

可发送：

```text
我的交易行
下架商品 商品ID
交易行订单 商品ID
```

`我的交易行` 查看自己上架的商品、库存、已售数量、剩余有效期和下架命令。`下架商品 商品ID` 下架自己的在售商品。背包商品下架后，未售库存会退回乾坤袋；自由卡密商品下架后，未售卡密不再出售。`交易行订单 商品ID` 查看自己某个商品最近成交订单。

### 买家订单与举报

可发送：

```text
我的购买
我的订单
举报订单 订单ID 原因
```

购买成功提示会显示订单号。`我的购买` 可查看最近购买记录和订单号，自由卡密可通过 `我的购买` 自助取回。背包物品订单显示已交付乾坤袋。举报原因 3-200 个字，不能包含换行、制表符或控制字符。

### 手续费与限制

交易行手续费：

```text
成交额的 3%
最低 1 积分
不会吃掉单笔全部成交额
成交额为 1 积分时手续费为 0
```

关键限制：

```text
自由上架：需筑基期及以上，单次最多 50 条卡密
卡密长度：每条最多 200 个字，且不能包含制表符或其他控制字符
商品名称：2-40 个字，不能包含换行、制表符或控制字符
背包物品名：1-40 个字，不能包含换行、制表符或控制字符
商品价格：1-100000 积分
背包上架数量：最多 1000
单次购买数量：最多 50
```

手续费会注入天道奖池。
